# Repo Catalog (Data Analytics & Data-Driven Problem Solving)

Checked: 2026-02-18

Use this list when you need a proven project structure, research template, or a lifecycle methodology.

| Repo | Focus | Use When | Notes |
| --- | --- | --- | --- |
| Azure/Azure-TDSP-ProjectTemplate | Team Data Science Process project template | You need a standardized analytics project structure with doc templates | Microsoft TDSP template repo |
| Azure/Microsoft-TDSP | TDSP methodology and lifecycle docs | You need a full lifecycle framework for analytics work | Archived; still useful for process guidance |
| Azure/Azure-TDSP-Utilities | TDSP utilities and helpers | You want supporting scripts and utilities for TDSP-style work | Archived; validate maintenance |
| drivendataorg/cookiecutter-data-science | Cookiecutter project scaffold | You need a flexible, widely used DS project layout | Mature template; fast start |
| the-turing-way/reproducible-project-template | Reproducible research template | You need a research-grade, reproducible structure | Good for academic-style research |
| JoseRZapata/data-science-project-template | Modern DS template with CI | You want packaging, tests, CI, and modern Python tooling | Good for production-oriented work |

Selection quick guide:
- Pick TDSP repos for end-to-end lifecycle framing and enterprise-style documentation.
- Pick Turing Way template for reproducible research and publication-ready structure.
- Pick cookiecutter-data-science for simple, flexible project scaffolding.
- Pick JoseRZapata template for modern Python tooling and CI from day one.
