# Module 1: การจัดการข้อมูลสุขภาพ (Data Management)

## หัวข้อการสอน

### 1.1 แหล่งข้อมูลสุขภาพไทย (3 ชั่วโมง)
- แหล่งข้อมูลภาครัฐ (MOPH, NHSO, NSO)
- แหล่งข้อมูลระหว่างประเทศ (WHO, World Bank)
- ข้อมูลเปิดภาครัฐ

### 1.2 การค้นหาและรวบรวมข้อมูล (2 ชั่วโมง)
- การกำหนดคำถามวิจัย (SMART Questions)
- การค้นหาข้อมูลเชิงกลยุทธ์
- การรวมข้อมูลจากหลายแหล่ง

### 1.3 การตรวจสอบความน่าเชื่อถือของข้อมูล (3 ชั่วโมง)
- กรอบ TRUST
- เทคนิค Cross-Validation
- การตรวจสอบคุณภาพข้อมูล
