| 8 | num_hospitals | จำนวนโรงพยาบาล | Integer | - | >=0 | 0 | จำนวนโรงพยาบาลทุกประเภท | MOPH |
| 9 | num_beds | จำนวนเตียง | Integer | - | >=0 | 0 | จำนวนเตียงรวมทั้งหมด | MOPH |

| 13 | bed_ratio | อัตราเตียง | Float | - | >=0 | - | จำนวนเตียงต่อประชากร 1,000 คน | คำนวณ |

| 15 | imr | อัตราตายทารก | Float | - | >=0 | - | อัตราตายทารกต่อ 1,000 เกิดมีชีพ | MOPH |

## ส่วนที่ 5: ความสัมพันธ์ระหว่างตัวแปร

### 5.1 Primary Key
- prov_code + year_data

### 5.2 Foreign Keys
- prov_code: เชื่อมโยงกับตาราง ref_provinces

### 5.3 Calculated Fields
| ตัวแปร | สูตรคำนวณ |
|--------|---------|
| density | population / area_km2 |
| doctor_ratio | (num_doctors / population) * 10000 |
| bed_ratio | (num_beds / population) * 1000 |
