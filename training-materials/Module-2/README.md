## ตัวอย่างการใช้ 5 Whys

**ปัญหา:** อัตราตายมารดาสูง

1. ทำไม? → มีภาวะแทรกซ้อนระหว่างตั้งครรภ์
2. ทำไม? → มารดาไม่ได้รับการดูแลก่อนคลอดที่เหมาะสม
3. ทำไม? → มารดาบางกลุ่มไม่เข้ารับบริการ ANC
4. ทำไม? → ไม่มีค่าเดินทางและขาดความรู้เรื่องสิทธิ
5. ทำไม? → ระบบข้อมูลไม่สามารถระบุกลุ่มเสี่ยงได้

**Root Cause:** ขาดระบบ early warning สำหรับมารดากลุ่มเสี่ยง
