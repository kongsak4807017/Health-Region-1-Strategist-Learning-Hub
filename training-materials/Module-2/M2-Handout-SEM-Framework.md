# Module 2.2: Social-Ecological Model (SEM) - กรอบการวิเคราะห์ระดับชุมชน
## เอกสารประกอบการสอน

---

## 1. ความเป็นมา

Social-Ecological Model (SEM) เป็นกรอบแนวคิดที่พัฒนาโดย Urie Bronfenbrenner ในปี 1979 สำหรับศึกษาพัฒนาการมนุษย์ ต่อมาถูกดัดแปลงและใช้อย่างกว้างขวางในด้านสาธารณสุขโดย CDC และ WHO เพื่อเข้าใจปัจจัยที่มีอิทธิพลต่อพฤติกรรมสุขภาพ

---

## 2. โครงสร้าง SEM

### 2.1 5 ระดับของ SEM

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  5. PUBLIC POLICY (นโยบายสาธารณะ)                           │
│     • กฎหมาย ระเบียบ                                       │
│     • การจัดสรรงบประมาณ                                    │
│     • นโยบายระดับชาติ                                      │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  4. COMMUNITY (ชุมชน)                                       │
│     • ความสัมพันธ์ในชุมชน                                  │
│     • นorms วัฒนธรรม                                       │
│     • สภาพแวดล้อมทางกายภาพ                                │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  3. ORGANIZATIONAL/INSTITUTIONAL (สถาบัน)                   │
│     • กฎระเบียบองค์กร                                      │
│     • วัฒนธรรมองค์กร                                       │
│     • ระบบสนับสนุน                                         │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  2. INTERPERSONAL (ระหว่างบุคคล)                            │
│     • ครอบครัว เพื่อน                                      │
│     • คู่สมรส                                              │
│     • เครือข่ายสังคม                                       │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. INDIVIDUAL (บุคคล)                                      │
│     • ความรู้ ทัศนคติ                                      │
│     • ทักษะ                                                │
│     • ประวัติสุขภาพ                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 ลักษณะสำคัญของแต่ละระดับ

| ระดับ | Focus | ปัจจัยสำคัญ | ตัวอย่างการแทรกแซง |
|-------|-------|-----------|-------------------|
| Individual | บุคคล | ความรู้ ทัศนคติ ทักษะ | ให้ความรู้ ฝึกอบรม |
| Interpersonal | ความสัมพันธ์ | ครอบครัว เพื่อน | สร้างเครือข่าย |
| Organizational | สถาบัน | กฎระเบียบ วัฒนธรรม | เปลี่ยน policy |
| Community | ชุมชน | norms สภาพแวดล้อม | สร้างความตระหนัก |
| Public Policy | นโยบาย | กฎหมาย การเงิน | รณรงค์เปลี่ยนกฎหมาย |

## 3. การใช้ SEM ในการวิเคราะห์ปัญหาสาธารณสุข

### 3.1 หลักการใช้งาน

```
1. ระบุปัญหาพฤติกรรม/สุขภาพที่ต้องการศึกษา
          ↓
2. วิเคราะห์ปัจจัยในแต่ละระดับของ SEM
          ↓
3. ระบุระดับที่เหมาะสมสำหรับการแทรกแซง
          ↓
4. ออกแบบ intervention ที่หลากหลายระดับ
```

### 3.2 ตัวอย่างการวิเคราะห์: การสูบบุหรี่ในวัยรุ่น

```
LEVEL 5: PUBLIC POLICY
├── กฎหมายห้ามขายบุหรี่ให้เยาวชน (ไม่บังคับใช้)
├── ภาษีบุหรี่ต่ำ
├── ไม่มีกฎหมายห้ามโฆษณาบุหรี่ครบวงจร
└── บุหรี่หาซื้อง่าย

LEVEL 4: COMMUNITY  
├── ชุมชนมองว่าสูบบุหรี่เป็นเรื่องปกติ
├── มีร้านขายบุหรี่ใกล้โรงเรียน
├── สื่อท้องถิ่นไม่รณรงค์ต่อต้านบุหรี่
└── ผู้ใหญ่ในชุมชนสูบบุหรี่เป็นแบบอย่าง

LEVEL 3: ORGANIZATIONAL
├── โรงเรียนไม่มีกฎห้ามสูบบุหรี่อย่างจริงจัง
├── ไม่มีนโยบายเชิงรุกป้องกันการสูบ
├── ครูบางคนสูบบุหรี่
└── ไม่มีกิจกรรมส่งเสริมสุขภาพ

LEVEL 2: INTERPERSONAL
├── เพื่อนสูบบุหรี่ (peer pressure)
├── พ่อแม่สูบบุหรี่
├── พี่น้องสูบบุหรี่
└── คู่รักสูบบุหรี่

LEVEL 1: INDIVIDUAL
├── ไม่รู้ถึงอันตรายของบุหรี่
├── อยากลองของ
├── คิดว่าสูบแล้วดูเท่
├── ความเครียด
└── ไม่มีทักษะการปฏิเสธ
```

### 3.3 การเลือกระดับการแทรกแซง

| ระดับ | Intervention | ประสิทธิผล | ต้นทุน | ความยาก |
|-------|-------------|----------|--------|---------|
| Individual | ให้ความรู้ | ต่ำ-ปานกลาง | ต่ำ | ง่าย |
| Interpersonal | ฝึกทักษะการปฏิเสธ | ปานกลาง | ต่ำ | ปานกลาง |
| Organizational | นโยบายโรงเรียนปลอดบุหรี่ | ปานกลาง-สูง | ปานกลาง | ปานกลาง |
| Community | รณรงค์ชุมชน | สูง | ปานกลาง | ยาก |
| Policy | กฎหมายควบคุมบุหรี่ | สูงมาก | สูง | ยากมาก |

**หลักการ:** Multi-level intervention มีประสิทธิผลสูงกว่า

## 4. ตัวอย่างการใช้ SEM กับปัญหาต่างๆ

### 4.1 ปัญหา: อัตราตายมารดาสูง

```
PUBLIC POLICY
├── ไม่มีมาตรฐาน ANC ที่ชัดเจน
├── ไม่มีงบประมาณสำหรับมารดากลุ่มเสี่ยง
└── ไม่มี incentive สำหรับแพทย์ทำงานพื้นที่ห่างไกล

COMMUNITY
├── ความเชื่อเรื่องการคลอดที่บ้าน
├── การตัดสินใจช้าในครอบครัว
├── ห่างไกลจาก รพ.
└── ค่าเดินทางสูง

ORGANIZATIONAL
├── รพ.สต. ไม่มีอุปกรณ์ตรวจครรภ์
├── ไม่มีระบบ refer ฉุกเฉิน
└── ไม่มีทีมสูติอุบัติเหตุ

INTERPERSONAL
├── สามีไม่ให้ไป รพ.
├── แม่สามีมีความเชื่อเดิม
└── ไม่มีญาติช่วยดูแลเด็ก

INDIVIDUAL
├── ไม่รู้จักสัญญาณอันตราย
├── ไม่เข้าใจความสำคัญของ ANC
└── กลัวค่าใช้จ่าย
```

### 4.2 ปัญหา: โรคเบาหวานไม่ควบคุม

```
PUBLIC POLICY
├── บัญชียาจำกัด
├── ไม่ครอบคลุมการตรวจ HbA1c
└── ไม่มีระบบ case management

COMMUNITY
├── อาหารขายในชุมชนไม่เหมาะสม
├── ไม่มีสนาม/สวนออกกำลังกาย
└── มองว่า DM เป็นเรื่องปกติ

ORGANIZATIONAL
├── รพ.สต. ไม่มีเครื่องตรวจน้ำตาล
├── ไม่มีเวลาให้คำแนะนำผู้ป่วย
└── ไม่มีระบบนัดหมาย/เตือน

INTERPERSONAL
├── ครอบครัวไม่ช่วยควบคุมอาหาร
├── เพื่อนชวนกินของหวาน
└── ไม่มีกลุ่มผู้ป่วย DM

INDIVIDUAL
├── ไม่เข้าใจโรค
├── ไม่มีวินัยทานยา
├── ไม่สามารถปฏิเสธอาหารได้
└── ไม่มีเวลาออกกำลังกาย
```

## 5. การออกแบบ Intervention ตาม SEM

### 5.1 ตัวอย่าง: โครงการลดการสูบบุหรี่ในวัยรุ่น

| ระดับ | Intervention | ผู้รับผิดชอบ | ตัวชี้วัด |
|-------|-------------|------------|----------|
| Policy | ออกกฎหมายห้ามขายบุหรี่ใกล้โรงเรียน | กระทรวงสาธารณสุข | % ร้านที่ปฏิบัติ |
| Community | รณรงค์ชุมชนปลอดบุหรี่ | อปท. | % ครัวเรือนปลอดบุหรี่ |
| Organizational | นโยบายโรงเรียนปลอดบุหรี่ | โรงเรียน | % นร. สูบลด |
| Interpersonal | ฝึกทักษะการปฏิเสธ | ครู/พยาบาล | ทักษะการปฏิเสธ |
| Individual | ให้ความรู้เรื่องบุหรี่ | ครู | ความรู้ |

### 5.2 การประเมินผลแต่ละระดับ

```
Level 1 (Individual):
- Pre/post test ความรู้
- ทัศนคติต่อการสูบบุหรี่

Level 2 (Interpersonal):
- Peer support network
- ความสัมพันธ์ในครอบครัว

Level 3 (Organizational):
- การบังคับใช้นโยบาย
- การเปลี่ยนแปลงวัฒนธรรม

Level 4 (Community):
- การเปลี่ยนแปลง norms
- การเข้าถึงบุหรี่

Level 5 (Policy):
- การบังคับใช้กฎหมาย
- การเปลี่ยนแปลงระดับชาติ
```

## 6. การใช้ SEM ร่วมกับเครื่องมืออื่น

### 6.1 SEM + Fishbone Diagram

```
Fishbone: ระบุสาเหตุตามหมวดหมู่ 4Ps
    ↓
SEM: จัดสาเหตุตามระดับ Individual → Policy
    ↓
เลือกระดับที่เหมาะสมสำหรับ intervention
```

### 6.2 SEM + Logic Model

```
SEM Level → Logic Model Component

Policy → Context/Inputs
Community → Activities
Organizational → Outputs
Interpersonal → Outcomes
Individual → Impact
```
