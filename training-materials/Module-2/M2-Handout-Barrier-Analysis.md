# Module 2.3: Barrier Analysis - การวิเคราะห์อุปสรรค
## เอกสารประกอบการสอน

---

## 1. ความหมายและความสำคัญ

**Barrier Analysis** คือ กระบวนการระบุและวิเคราะห์อุปสรรคที่ขัดขวางไม่ให้กลุ่มเป้าหมายเข้าถึงบริการหรือเปลี่ยนแปลงพฤติกรรม เป็นเครื่องมือสำคัญสำหรับการออกแบบ intervention ที่ตอบโจทย์ความต้องการจริง

---

## 2. ประเภทของ Barriers

### 2.1 ตามลักษณะ

| ประเภท | อธิบาย | ตัวอย่าง |
|--------|--------|---------|
| **Physical** | อุปสรรคทางกายภาพ | ระยะทาง การคมนาคม |
| **Financial** | อุปสรรคทางการเงิน | ค่าใช้จ่าย ค่าเดินทาง |
| **Informational** | อุปสรรคด้านข้อมูล | ไม่รู้ว่ามีบริการ ไม่รู้วิธีใช้ |
| **Attitudinal** | อุปสรรคทางทัศนคติ | ความกลัว ความเชื่อ |
| **Social** | อุปสรรคทางสังคม | แรงกดดัน การตีตรา |
| **Structural** | อุปสรรบเชิงโครงสร้าง | เวลาทำการ ขั้นตอน |

### 2.2 ตามระดับ

```
ระดับโครงสร้าง (Structural)
    ↓
ระดับชุมชน (Community)  
    ↓
ระดับครอบครัว (Household)
    ↓
ระดับบุคคล (Individual)
```

## 3. ขั้นตอนการทำ Barrier Analysis

### Step 1: กำหนด Behavior ที่ต้องการส่งเสริม

**ตัวอย่าง:**
- หญิงตั้งครรภ์เข้ารับ ANC 5 ครั้ง
- ผู้ป่วยเบาหวานตรวจ HbA1c ทุก 3 เดือน
- เด็กได้รับวัคซีนครบตามเกณฑ์

### Step 2: ระบุกลุ่มเป้าหมาย

แบ่งกลุ่มเป้าหมายออกเป็น:
- **Doers:** คนที่ทำพฤติกรรมที่ต้องการแล้ว
- **Non-doers:** คนที่ไม่ทำพฤติกรรมที่ต้องการ

### Step 3: ระบุ Barriers สำหรับ Non-doers

**เทคนิคการเก็บข้อมูล:**
- In-depth interview
- Focus group discussion
- Observation
- Review records

### Step 4: วิเคราะห์ Barriers

จัดกลุ่มและจัดลำดับความสำคัญ

### Step 5: ออกแบบ Intervention

ออกแบบการแก้ไขตาม barrier ที่พบ


### 4.2 คำถามสัมภาษณ์เชิงลึก

**สำหรับ Doers:**
1. อะไรที่ทำให้คุณตัดสินใจทำ...?
2. มีอุปสรรคอะไรบ้างที่คุณเผชิญ? แล้วคุณเอาชนะได้อย่างไร?
3. อะไรที่ช่วยสนับสนุนให้คุณทำสำเร็จ?

**สำหรับ Non-doers:**
1. ทำไมคุณถึงยังไม่ได้ทำ...?
2. มีอะไรที่ทำให้คุณไม่สามารถทำได้บ้าง?
3. ถ้าจะให้คุณทำได้ ต้องมีอะไรเปลี่ยนแปลงบ้าง?
4. อะไรที่จะทำให้คุณเริ่มทำ...?

## 5. ตัวอย่าง Barrier Analysis

### 5.1 ตัวอย่างที่ 1: การเข้ารับ ANC

**Behavior:** หญิงตั้งครรภ์เข้ารับ ANC 5 ครั้ง

**กลุ่มเป้าหมาย:**
- Doers: หญิงที่เข้า ANC ครบ 5 ครั้ง
- Non-doers: หญิงที่เข้า ANC น้อยกว่า 5 ครั้ง

**Barriers ที่พบ:**

| Barrier | % ของ Non-doers | ระดับ |
|---------|----------------|-------|
| ไม่รู้ว่าต้องมา 5 ครั้ง | 35% | Individual |
| ห่างไกลจาก รพ. | 28% | Physical |
| ไม่มีค่าเดินทาง | 25% | Financial |
| สามีไม่ให้มา | 18% | Social |
| รอคิวนาน | 15% | Structural |
| กลัวถูกตำหนิ | 12% | Attitudinal |

**Intervention ที่ออกแบบ:**

| Barrier | Intervention |
|---------|-------------|
| ไม่รู้ว่าต้องมา 5 ครั้ง | ให้ข้อมูลผ่าน อสม. |
| ห่างไกล | ตั้งจุดบริการเคลื่อนที่ |
| ไม่มีค่าเดินทาง | จัดสรรค่าเดินทาง |
| สามีไม่ให้มา | ให้คำปรึกษาคู่สมรส |
| รอคิวนาน | นัดหมายล่วงหน้า |
| กลัวถูกตำหนิ | อบรมบุคลากรเชิงบวก |

### 5.2 ตัวอย่างที่ 2: การควบคุมเบาหวาน

**Behavior:** ผู้ป่วย DM ตรวจ HbA1c ทุก 3 เดือน

**Barriers ที่พบจากการสัมภาษณ์:**

**Physical:**
- รพ.สต. ไม่มีเครื่องตรวจ
- ต้องไป รพ.ใหญ่ ห่าง 40 กม.

**Financial:**
- HbA1c ไม่ครอบคลุม UCS ทุกครั้ง
- ค่าเดินทางสูง

**Informational:**
- ไม่รู้ว่าต้องตรวจบ่อยแค่ไหน
- ไม่เข้าใจค่า HbA1c

**Attitudinal:**
- รู้สึกสบายดี ไม่คิดว่าต้องตรวจ
- กลัวผลตรวจ

**Social:**
- ครอบครัวไม่เตือน
- เพื่อนบ้านบอกว่า "แก่แล้วเป็นอย่างนี้แหละ"

**Structural:**
- รพ. นัดยาก
- ต้องใช้เวลาทั้งวัน

## 6. การวิเคราะห์และจัดลำดับ Barriers

### 6.1 ตารางวิเคราะห์

| Barrier | ความถี่ | ความรุนแรง | ความง่ายในการแก้ไข | ลำดับความสำคัญ |
|---------|--------|-----------|-------------------|---------------|
| | สูง/กลาง/ต่ำ | สูง/กลาง/ต่ำ | ง่าย/ยาก | |

### 6.2 Matrix การตัดสินใจ

```
            ความง่ายในการแก้ไข
              ง่าย        ยาก
           ┌─────────┬─────────┐
    สูง    │ ทำก่อน  │ วางแผน  │
ความรุนแรง │ Priority│ ระยะยาว│
           ├─────────┼─────────┤
    ต่ำ    │ ทำถ้ามี │ ไม่ทำ  │
           │ เวลา    │ ก่อน    │
           └─────────┴─────────┘
```

## 7. การออกแบบ Intervention ตาม Barrier

### 7.1 หลักการ

```
Barrier ที่พบ → Intervention ที่เหมาะสม
```

### 7.2 ตัวอย่าง Intervention

| Barrier Type | Intervention Examples |
|-------------|----------------------|
| Physical | ตั้งจุดบริการใกล้บ้าน, รถบริการเคลื่อนที่ |
| Financial | ให้เงินช่วยเหลือ, ยกเว้นค่าธรรมเนียม |
| Informational | ให้ข้อมูล, ใช้ อสม. ให้คำแนะนำ |
| Attitudinal | ให้คำปรึกษา, ใช้ role model |
| Social | ทำงานกับครอบครัว, สร้างกลุ่มสนับสนุน |
| Structural | ขยายเวลาทำการ, ลดขั้นตอน |
