# Module 3.3: Multi-Criteria Decision Analysis (MCDA)
## คู่มือการเปรียบเทียบทางเลือกนโยบาย

---

## 1. ความหมายและความสำคัญ

**Multi-Criteria Decision Analysis (MCDA)** เป็นเทคนิคการตัดสินใจที่ช่วยให้ผู้กำหนดนโยบายสามารถเปรียบเทียบทางเลือกต่างๆ โดยพิจารณาหลายเกณฑ์พร้อมกัน ช่วยให้การตัดสินใจเป็นระบบ โปร่งใส และสามารถอธิบายได้

**ทำไมต้องใช้ MCDA?**
- นโยบายสุขภาพมีหลายมิติ (ประสิทธิผล ความเป็นธรรม ต้นทุน)
- ผู้มีส่วนได้ส่วนเสียมีความสนใจต่างกัน
- ต้องการความโปร่งใสในการตัดสินใจ
- ต้องการสร้างฉันทามติ

---

## 2. ขั้นตอนการทำ MCDA

```
┌─────────────────────────────────────────────────────────────┐
│  STEP 1: กำหนดทางเลือก (Alternatives)                       │
│  ระบุทางเลือกนโยบายที่เป็นไปได้                             │
├─────────────────────────────────────────────────────────────┤
│  STEP 2: กำหนดเกณฑ์ (Criteria)                              │
│  เลือกเกณฑ์ที่ใช้ประเมิน                                    │
├─────────────────────────────────────────────────────────────┤
│  STEP 3: ให้น้ำหนักเกณฑ์ (Weighting)                        │
│  กำหนดความสำคัญของแต่ละเกณฑ์                               │
├─────────────────────────────────────────────────────────────┤
│  STEP 4: ให้คะแนน (Scoring)                                 │
│  ประเมินแต่ละทางเลือกตามเกณฑ์                              │
├─────────────────────────────────────────────────────────────┤
│  STEP 5: คำนวณและวิเคราะห์ (Calculation)                    │
│  คำนวณคะแนนรวมและเปรียบเทียบ                               │
├─────────────────────────────────────────────────────────────┤
│  STEP 6: ตรวจสอบความไว (Sensitivity Analysis)              │
│  ทดสอบว่าผลเปลี่ยนหากเกณฑ์เปลี่ยน                          │
└─────────────────────────────────────────────────────────────┘
```

---

## 3. STEP 1: กำหนดทางเลือก (Alternatives)

### 3.1 หลักการเลือกทางเลือก

- ต้องเป็นไปได้จริง (feasible)
- ครอบคลุมขอบเขตกว้าง (range of options)
- ไม่ซ้ำซ้อนกัน (mutually exclusive)

### 3.2 ตัวอย่างทางเลือกนโยบาย

**ปัญหา:** ลดการสูบบุหรี่ในวัยรุ่น

| ทางเลือก | รายละเอียด |
|---------|-----------|
| **A** | ห้ามโฆษณาบุหรี่ทุกชนิด |
| **B** | เพิ่มภาษีบุหรี่ 20% |
| **C** | รณรงค์ในโรงเรียน |
| **D** | รวม A+B+C |
| **E** | ไม่ทำอะไร (status quo) |

---

## 4. STEP 2: กำหนดเกณฑ์ (Criteria)

### 4.1 เกณฑ์มาตรฐานสำหรับนโยบายสุขภาพ

| เกณฑ์ | คำอธิบาย | น้ำหนักแนะนำ |
|-------|---------|-------------|
| **Effectiveness** | ประสิทธิผลในการแก้ปัญหา | 25% |
| **Cost** | ต้นทุน/งบประมาณที่ต้องใช้ | 20% |
| **Equity** | ความเป็นธรรม ลดความเหลื่อมล้ำ | 15% |
| **Feasibility** | ความเป็นไปได้ในการดำเนินการ | 15% |
| **Acceptability** | ความยอมรับจากผู้มีส่วนได้ส่วนเสีย | 10% |
| **Sustainability** | ความยั่งยืนในระยะยาว | 10% |
| **Safety** | ความปลอดภัย/ความเสี่ยง | 5% |

### 4.2 ตัวอย่างเกณฑ์เฉพาะด้าน

**ด้านสาธารณสุข:**
- ลดอัตราป่วย/ตาย
- ปรับปรุงคุณภาพชีวิต (QALYs)
- ครอบคลุมประชากร

**ด้านเศรษฐกิจ:**
- ต้นทุน/ประโยชน์ (Cost-benefit)
- คุ้มค่า (Cost-effectiveness)
- ผลกระทบทางเศรษฐกิจ

**ด้านสังคม:**
- ความพึงพอใจ
- ผลกระทบต่อกลุ่มเปราะบาง
- ความเป็นส่วนตัว

---

## 5. STEP 3: ให้น้ำหนักเกณฑ์ (Weighting)

### 5.1 วิธีการให้น้ำหนัก

**วิธีที่ 1: Direct Rating**
- ให้คะแนนความสำคัญของแต่ละเกณฑ์โดยตรง
- รวมกัน = 100%

**วิธีที่ 2: Pairwise Comparison**
- เปรียบเทียบเกณฑ์คู่กัน
- ใช้วิธี Analytic Hierarchy Process (AHP)

### 5.2 ตัวอย่างการให้น้ำหนัก

```
การให้น้ำหนักโดยผู้มีส่วนได้ส่วนเสียหลายกลุ่ม:

┌──────────────────┬──────────┬──────────┬──────────┬──────────┐
│    เกณฑ์         │ กระทรวง │ ผู้ป่วย  │ นักวิชา │ เฉลี่ย  │
│                  │   สธ.    │         │  การ    │         │
├──────────────────┼──────────┼──────────┼──────────┼──────────┤
│ Effectiveness    │   30%    │   35%    │   25%    │   30%   │
│ Cost             │   20%    │   10%    │   25%    │   18%   │
│ Equity           │   15%    │   25%    │   20%    │   20%   │
│ Feasibility      │   20%    │   15%    │   15%    │   17%   │
│ Acceptability    │   10%    │   10%    │   10%    │   10%   │
│ Sustainability   │    5%    │    5%    │    5%    │    5%   │
├──────────────────┼──────────┼──────────┼──────────┼──────────┤
│ รวม             │  100%    │  100%    │  100%    │  100%   │
└──────────────────┴──────────┴──────────┴──────────┴──────────┘
```

## 6. STEP 4: ให้คะแนน (Scoring)

### 6.1 เกณฑ์การให้คะแนน

| คะแนน | ระดับ | คำอธิบาย |
|-------|-------|---------|
| 5 | ดีมาก (Excellent) | เกินเป้าหมาย |
| 4 | ดี (Good) | ตามเป้าหมาย |
| 3 | ปานกลาง (Fair) | พอใช้ |
| 2 | ต่ำ (Poor) | ต้องปรับปรุง |
| 1 | แย่ (Very Poor) | ไม่เหมาะสม |

### 6.2 ตัวอย่างการให้คะแนน

**ปัญหา:** ลดการสูบบุหรี่ในวัยรุ่น

| เกณฑ์ | น้ำหนัก | A: ห้ามโฆษณา | B: เพิ่มภาษี | C: รณรงค์โรงเรียน |
|-------|--------|------------|------------|------------------|
| Effectiveness | 30% | 4 | 3 | 3 |
| Cost | 18% | 5 | 5 | 4 |
| Equity | 20% | 5 | 2 | 3 |
| Feasibility | 17% | 3 | 2 | 5 |
| Acceptability | 10% | 2 | 1 | 5 |
| Sustainability | 5% | 4 | 3 | 4 |

**คำอธิบายคะแนน:**
- **A ห้ามโฆษณา:**
  - Effectiveness 4 = มีงานวิจัยสนับสนุนว่าลดการสูบได้
  - Cost 5 = ใช้งบน้อยกว่าการรณรงค์
  - Acceptability 2 = อุตสาหกรรมคัดค้าน

- **B เพิ่มภาษี:**
  - Equity 2 = กระทบคนจนมากกว่า
  - Acceptability 1 = คนสูบคัดค้านแรง

- **C รณรงค์โรงเรียน:**
  - Feasibility 5 = ทำได้ง่าย
  - Acceptability 5 = ผู้ปกครองสนับสนุน

---

## 7. STEP 5: คำนวณและวิเคราะห์

### 7.1 การคำนวณคะแนนรวม

```
คะแนนรวม = Σ (คะแนน × น้ำหนัก)

ตัวอย่างการคำนวณทางเลือก A (ห้ามโฆษณา):

Effectiveness: 4 × 30% = 1.20
Cost:          5 × 18% = 0.90
Equity:        5 × 20% = 1.00
Feasibility:   3 × 17% = 0.51
Acceptability: 2 × 10% = 0.20
Sustainability:4 × 5%  = 0.20
────────────────────────────────
รวม                          = 4.01
```

### 7.2 ตารางสรุปผล

| ทางเลือก | Effectiveness (30%) | Cost (18%) | Equity (20%) | Feasibility (17%) | Acceptability (10%) | Sustainability (5%) | รวม |
|---------|-------------------|-----------|-------------|------------------|-------------------|-------------------|-----|
| A ห้ามโฆษณา | 1.20 | 0.90 | 1.00 | 0.51 | 0.20 | 0.20 | **4.01** |
| B เพิ่มภาษี | 0.90 | 0.90 | 0.40 | 0.34 | 0.10 | 0.15 | **2.79** |
| C รณรงค์ | 0.90 | 0.72 | 0.60 | 0.85 | 0.50 | 0.20 | **3.77** |

**ผลลัพธ์:**
1. 🥇 A (ห้ามโฆษณา) = 4.01
2. 🥈 C (รณรงค์) = 3.77
3. 🥉 B (เพิ่มภาษี) = 2.79

### 7.3 การวิเคราะห์เพิ่มเติม

**จุดแข็ง/จุดอ่อนแต่ละทางเลือก:**

| ทางเลือก | จุดแข็ง | จุดอ่อน |
|---------|--------|--------|
| A | ประสิทธิผลสูง ต้นทุนต่ำ | อาจถูกท้าทายทางกฎหมาย |
| B | เก็บรายได้ | ไม่เป็นธรรม |
| C | ยอมรับได้ง่าย | ผลช้า |

## 8. STEP 6: Sensitivity Analysis

### 8.1 ความหมาย

ทดสอบว่าผลการตัดสินใจเปลี่ยนไหม หากน้ำหนักเกณฑ์เปลี่ยน

### 8.2 ตัวอย่าง

**สถานการณ์ที่ 1:** ให้ความสำคัญกับ Equity มากขึ้น (30%)

| ทางเลือก | คะแนนใหม่ | ลำดับ |
|---------|----------|-------|
| A | 4.31 | 1 |
| B | 2.49 | 3 |
| C | 3.77 | 2 |

**สถานการณ์ที่ 2:** ให้ความสำคัญกับ Acceptability มากขึ้น (30%)

| ทางเลือก | คะแนนใหม่ | ลำดับ |
|---------|----------|-------|
| A | 3.41 | 2 |
| B | 2.59 | 3 |
| C | 4.37 | 1 |

**สรุป:** A ยังคงเป็นทางเลือกที่ดีที่สุดในสถานการณ์ส่วนใหญ่

## 9. SWOT Analysis ร่วมกับ MCDA

### 9.1 การใช้ SWOT กับทางเลือกที่เลือก

**ทางเลือกที่เลือก: A (ห้ามโฆษณา)**

| | ช่วย (Helpful) | อุปสรรค (Harmful) |
|---|---------------|------------------|
| **ภายใน (Internal)** | **Strengths** | **Weaknesses** |
| | • มีงานวิจัยสนับสนุน | • ต้องมีการบังคับใช้ |
| | • ต้นทุนต่ำ | • อาจถูกท้าทาย |
| **ภายนอก (External)** | **Opportunities** | **Threats** |
| | • สื่อสนับสนุน | • อุตสาหกรรมต่อต้าน |
| | • ประชาชนสนับสนุน | • การล็อบบี้ |

### 9.2 การนำผลไปใช้

จาก SWOT ออกแบบกลยุทธ์:
- **SO Strategy:** ใช้ข้อมูลวิจัยรณรงค์กับสื่อ
- **WO Strategy:** สร้างกลไกบังคับใช้กฎหมาย
- **ST Strategy:** เตรียมข้อมูลตอบโต้อุตสาหกรรม
- **WT Strategy:** หาพันธมิตรสนับสนุน

## 10. Scenario Planning

### 10.1 การวางแผนรองรับสถานการณ์

| สถานการณ์ | ความเป็นไปได้ | แนวทางการดำเนินการ |
|----------|--------------|-------------------|
| **Best Case** | รัฐบาลสนับสนุน อุตสาหกรรมยอมรับ | ดำเนินการเต็มรูปแบบ |
| **Most Likely** | รัฐบาลสนับสนุน แต่มีความต้านทาน | ทยอยดำเนินการ พร้อมประนีประนอม |
| **Worst Case** | รัฐบาลเปลี่ยน ถูกท้าทายทางกฎหมาย | รอจังหวะ รณรงค์ต่อเนื่อง |

## แบบฝึกหัด

### แบบฝึกหัดที่ 1: ทำ MCDA

เลือกหนึ่งปัญหาแล้วทำ MCDA:
- [ ] การลดอัตราตายมารดา
- [ ] การเพิ่มการฝากครรภ์
- [ ] การควบคุมโรคเบาหวาน
- [ ] การลดการสูบบุหรี่

### แบบฝึกหัดที่ 2: Sensitivity Analysis

จากผล MCDA ที่ได้:
1. เปลี่ยนน้ำหนักเกณฑ์ 2 สถานการณ์
2. ดูว่าผลเปลี่ยนหรือไม่
3. สรุปว่าทางเลือกที่เลือกมีความมั่นคงหรือไม่
