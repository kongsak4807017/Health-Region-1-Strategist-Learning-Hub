# Module 3.2: Stakeholder Analysis - การวิเคราะห์ผู้มีส่วนได้ส่วนเสีย
## เอกสารประกอบการสอน

---

## 1. ความหมายและความสำคัญ

**Stakeholder** คือ บุคคล กลุ่ม หรือองค์กรที่มีผลประโยชน์เกี่ยวข้อง (interest) หรือได้รับผลกระทบ (affected) จากนโยบาย โครงการ หรือการตัดสินใจใดๆ

**ความสำคัญของ Stakeholder Analysis:**
- รู้จักผู้ที่มีอิทธิพลต่อความสำเร็จของนโยบาย
- เข้าใจผลประโยชน์และแรงจูงใจของแต่ละฝ่าย
- วางแผนกลยุทธ์การสื่อสารและการมีส่วนร่วม
- ลดความขัดแย้งและเพิ่มการสนับสนุน

## 2. ประเภทของ Stakeholders

### 2.1 ตามลักษณะ

| ประเภท | คำอธิบาย | ตัวอย่าง |
|--------|---------|---------|
| **Primary** | ได้รับผลกระทบโดยตรง | ผู้ป่วย ผู้รับบริการ |
| **Secondary** | ได้รับผลกระทบทางอ้อม | ครอบครัวผู้ป่วย |
| **Key** | มีอำนาจตัดสินใจ | รัฐมนตรี ผู้บริหาร |
| **Interest** | สนใจ/มีส่วนได้เสีย | สมาคมวิชาชีพ |

## 3. กรอบการวิเคราะห์ Stakeholder

### 3.1 เกณฑ์การวิเคราะห์

| เกณฑ์ | คำอธิบาย | ระดับ |
|-------|---------|-------|
| **Interest** | ความสนใจ/ผลประโยชน์ที่เกี่ยวข้อง | สูง / กลาง / ต่ำ |
| **Influence** | อำนาจต่อรอง/กระทบต่อความสำเร็จ | สูง / กลาง / ต่ำ |
| **Position** | ท่าทีต่อนโยบาย | สนับสนุน / เป็นกลาง / คัดค้าน |
| **Resource** | ทรัพยากรที่สามารถให้ได้ | เงิน บุคลากร ข้อมูล |

### 3.2 Stakeholder Matrix

| Stakeholder | Interest | Influence | Position | Priority |
|------------|----------|-----------|----------|----------|
| กระทรวงสาธารณสุข | สูง | สูง | สนับสนุน | สูง |
| สปสช. | สูง | สูง | เป็นกลาง | สูง |
| รพ.เอกชน | ปานกลาง | ปานกลาง | คัดค้าน | ปานกลาง |
| ผู้ป่วย | สูง | ต่ำ | สนับสนุน | ปานกลาง |
| สมาคมแพทย์ | สูง | ปานกลาง | เป็นกลาง | ปานกลาง |

## 4. Power/Interest Grid

### 4.1 โครงสร้าง

```
           Interest (ความสนใจ)
              ต่ำ          สูง
              |            |
    สูง ------+------------+--------- 
      Keep    |   Manage   |
    Satisfied |   Closely  |
              |            |
Influence     |            |
(อำนาจ)       |            |
              |            |
    ต่ำ ------+------------+---------
      Monitor |   Keep     |
              |   Informed |
              |            |
```

### 4.2 กลยุทธ์การจัดการแต่ละกลุ่ม

| กลุ่ม | ลักษณะ | กลยุทธ์ | ตัวอย่าง |
|------|--------|--------|---------|
| **Manage Closely** | อำนาจสูง สนใจสูง | ร่วมมืออย่างใกล้ชิด สร้างพันธมิตร | กระทรวงสธ., สปสช. |
| **Keep Satisfied** | อำนาจสูง สนใจต่ำ | รักษาความพึงพอใจ ไม่ให้ต่อต้าน | รัฐบาล สนช. |
| **Keep Informed** | อำนาจต่ำ สนใจสูง | ให้ข้อมูล สร้างการมีส่วนร่วม | ผู้ป่วย ชุมชน |
| **Monitor** | อำนาจต่ำ สนใจต่ำ | ติดตาม ไม่ใช้ทรัพยากรมาก | ประชาชนทั่วไป |

### 4.3 ตัวอย่าง Power/Interest Grid: นโยบาย UCS

```
              Interest (ความสนใจ)
                 ต่ำ          สูง
                 |            |
       สูง ------+------------+---------
                 |  Manage    |
    รัฐบาล       |  Closely   |  กระทรวงสธ.
    (Keep        |            |  สปสช.
    Satisfied)   |            |
                 |            |
Influence        |            |
                 |            |
       ต่ำ ------+------------+---------
                 |  Keep      |
    ประชาชนทั่วไป|  Informed  |  ผู้ป่วย
    (Monitor)    |            |  กลุ่มผู้ป่วย
                 |            |
```

## 5. ขั้นตอนการทำ Stakeholder Analysis

### Step 1: ระบุ Stakeholders ทั้งหมด
Brainstorm รายชื่อผู้มีส่วนได้ส่วนเสียที่อาจเกี่ยวข้อง

### Step 2: วิเคราะห์แต่ละเกณฑ์
ประเมิน Interest, Influence, Position, Resource

### Step 3: จัดกลุ่มใน Power/Interest Grid
วางตำแหน่งแต่ละ stakeholder ในกริด

### Step 4: กำหนดกลยุทธ์
เลือกกลยุทธ์การจัดการตามกลุ่ม

### Step 5: วางแผนการสื่อสาร
กำหนดวิธี ความถี่ และผู้รับผิดชอบ

### Step 6: ทบทวนและปรับปรุง
ติดตามการเปลี่ยนแปลงและปรับกลยุทธ์

## 6. กลยุทธ์การมีส่วนร่วม

### 6.1 Spectrum of Participation

```
┌─────────────────────────────────────────────────────────────┐
│           SPECTRUM OF PUBLIC PARTICIPATION                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Inform ← Consult ← Involve ← Collaborate ← Empower        │
│                                                             │
│  แจ้งให้รู้   ปรึกษา   มีส่วนร่วม   ร่วมมือ   มอบอำนาจ     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 การเลือกระดับการมีส่วนร่วม

| Stakeholder | ระดับ | วิธีการ |
|------------|------|--------|
| กระทรวงสธ. | Collaborate | ประชุมร่วม ตัดสินใจร่วม |
| สปสช. | Collaborate | Task force ร่วม |
| ผู้ป่วย | Involve | Focus group ร่วมออกแบบ |
| ประชาชน | Consult | สำรวจความคิดเห็น |
| สื่อ | Inform | แถลงข่าว ให้สัมภาษณ์ |

### 6.3 แผนการสื่อสาร

| Stakeholder | วิธีการ | ความถี่ | ผู้รับผิดชอบ |
|------------|--------|---------|-------------|
| กระทรวงสธ. | ประชุมผู้บริหาร | รายเดือน | ผอ.กอง |
| สปสช. | คณะทำงานร่วม | รายสัปดาห์ | ทีมงาน |
| ผู้ป่วย | Focus group | รายไตรมาส | พยาบาล |
| สื่อ | แถลงข่าว | ตาม milestones | โฆษก |

## 7. การจัดการความขัดแย้ง

### 7.1 ประเภทของความขัดแย้ง

| ประเภท | คำอธิบาย | ตัวอย่าง |
|--------|---------|---------|
| **ผลประโยชน์** | ได้/เสียจากนโยบาย | รพ.เอกชนกลัวรายได้ลด |
| **ค่านิยม** | ความเชื่อต่างกัน | การทำแท้ง |
| **ข้อมูล** | มีข้อมูลไม่ตรงกัน | ตัวเลขค่าใช้จ่าย |
| **ความสัมพันธ์** | ปัญหาส่วนตัว | ความไม่ลงรอยกันระหว่างหน่วยงาน |

### 7.2 กลยุทธ์การจัดการ

| กลยุทธ์ | เหมาะกับ | วิธีการ |
|--------|---------|--------|
| **Compete** | ต้องการชนะ | ใช้อำนาจตัดสินใจ |
| **Collaborate** | ผลประโยชน์ร่วม | หาทางออกร่วม (win-win) |
| **Compromise** | ต้องการแก้ปัญหา | ให้และรับ (negotiate) |
| **Avoid** | ไม่สำคัญ | เลี่ยงหรือเลื่อน |
| **Accommodate** | รักษาความสัมพันธ์ | ยอมตาม |

### 7.3 การเจรจา (Negotiation)

**หลักการเจรจากับ Stakeholder:**

1. **เตรียมตัว:**
   - รู้จุดยืนของตนเอง (BATNA)
   - รู้ผลประโยชน์ของอีกฝ่าย
   - หาข้อมูลสนับสนุน

2. **ระหว่างเจรจา:**
   - ฟังก่อน พูดทีหลัง
   - หาจุดร่วม
   - เสนอทางเลือก

3. **หลังเจรจา:**
   - สรุปข้อตกลงเป็นลายลักษณ์อักษร
   - ติดตามการปฏิบัติ

## 8. ตัวอย่าง Stakeholder Analysis

### 8.1 นโยบาย: ยกระดับค่าตอบแทนบุคลากรสาธารณสุข

| Stakeholder | Interest | Influence | Position | Strategy |
|------------|----------|-----------|----------|----------|
| สธ. | สูง | สูง | สนับสนุน | ร่วมมือ |
| สำนักงบฯ | สูง | สูง | คัดค้าน | เจรจา |
| ข้าราชการสธ. | สูง | กลาง | สนับสนุน | ให้ข้อมูล |
| รพ.เอกชน | กลาง | กลาง | เป็นกลาง | ติดตาม |
| ผู้ป่วย | กลาง | ต่ำ | สนับสนุน | ติดตาม |

**วิเคราะห์:**
- **สำนักงบฯ:** คัดค้านเพราะต้องใช้งบมาก ต้องเจรจาหาทางลดต้นทุนอื่น
- **ข้าราชการสธ.:** สนับสนุนแต่ไม่มีอำนาจตัดสินใจ ต้องให้ข้อมูลสนับสนุน

### 8.2 นโยบาย: ควบคุมการโฆษณาอาหารเด็ก

| Stakeholder | Interest | Influence | Position | Strategy |
|------------|----------|-----------|----------|----------|
| สธ. | สูง | สูง | สนับสนุน | ร่วมมือ |
| อุตสาหกรรมอาหาร | สูง | สูง | คัดค้าน | เจรจา |
| ผู้ปกครอง | สูง | กลาง | สนับสนุน | ให้ข้อมูล |
| สื่อ | กลาง | กลาง | เป็นกลาง | โน้มน้าว |

**วิเคราะห์:**
- **อุตสาหกรรมอาหาร:** คัดค้านแรงเพราะกระทบรายได้ ต้องเจรจาหา compromise
- **สื่อ:** เป็นกลางเพราะกระทบรายได้โฆษณา ต้องโน้มน้าวด้วยข้อมูลผลกระทบสุขภาพ
