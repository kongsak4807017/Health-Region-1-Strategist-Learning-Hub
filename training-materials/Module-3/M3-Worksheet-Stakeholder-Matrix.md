## ส่วนที่ 3: Stakeholder Matrix

### 3.1 วิเคราะห์แต่ละ Stakeholder

เลือก Stakeholders สำคัญ 6-8 ราย แล้วกรอกข้อมูล:

| Stakeholder | Interest (ความสนใจ) | Influence (อำนาจ) | Position (ตำแหน่ง) | Resource (ทรัพยากร) |
|------------|-------------------|------------------|-------------------|-------------------|
| | สูง/กลาง/ต่ำ | สูง/กลาง/ต่ำ | สนับสนุน/เป็นกลาง/คัดค้าน | เงิน/บุคลากร/ข้อมูล/อำนาจตัดสินใจ |
| | | | | |
| | | | | |
| | | | | |
| | | | | |
| | | | | |
| | | | | |
| | | | | |

**คำอธิบายเกณฑ์:**
- **Interest:** ผลประโยชน์ที่ได้รับหรือเสียหายจากนโยบาย
- **Influence:** ความสามารถในการส่งผลต่อความสำเร็จของนโยบาย
- **Position:** ท่าทีต่อนโยบาย (อาจเปลี่ยนแปลงได้)
- **Resource:** ทรัพยากรที่สามารถนำมาใช้ได้
