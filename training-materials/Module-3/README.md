# Module 3: กรอบการวิเคราะห์นโยบาย (Policy Analysis)

## หัวข้อการสอน

### 3.1 ทฤษฎีและกรอบการวิเคราะห์นโยบาย (3 ชั่วโมง)
- Policy Triangle Framework (Walt & Gilson)
- Policy Cycle
- Health in All Policies (HiAP)

### 3.2 การวิเคราะห์ผู้มีส่วนได้ส่วนเสีย (2 ชั่วโมง)
- การระบุ Stakeholders
- Stakeholder Analysis Matrix
- Power/Interest Grid
- กลยุทธ์การมีส่วนร่วม

### 3.3 การเปรียบเทียบทางเลือกนโยบาย (3 ชั่วโมง)
- SWOT Analysis
- Multi-Criteria Decision Analysis (MCDA)
- Cost-Effectiveness Analysis (เบื้องต้น)
- Scenario Planning

### Activity 1: Policy Triangle Analysis

### Activity 3: MCDA Workshop
