# Module 3.1: Policy Triangle Framework (Walt & Gilson)
## เอกสารประกอบการสอน - ฉบับละเอียด

---

## 1. ความเป็นมา

**Policy Triangle Framework** เป็นกรอบการวิเคราะห์นโยบายที่พัฒนาโดย Walt และ Gilson ในปี 1994 โดยมีวัตถุประสงค์เพื่อเข้าใจปัจจัยหลายมิติที่มีผลต่อการกำหนดและดำเนินนโยบายสุขภาพใน LMICs (Low and Middle Income Countries) รวมถึงประเทศไทย

---

## 2. โครงสร้าง Policy Triangle

### 2.1 รูปแบบพื้นฐาน

```
                Content (เนื้อหา)
                     /\
                    /  \
                   /    \
                  /______\
    Context (บริบท)        Process (กระบวนการ)
```

### 2.2 ส่วนประกอบทั้งสี่

| ส่วน | คำถามหลัก | ตัวอย่าง |
|-----|----------|---------|
| **Content** | นโยบายว่าด้วยเรื่องใด? | หลักประกันสุขภาพถ้วนหน้า |
| **Context** | บริบทเป็นอย่างไร? | การเมือง เศรษฐกิจ สังคม |
| **Process** | กระบวนการเป็นอย่างไร? | ร่าง ผลักดัน ตัดสินใจ |
| **Actors** | ใครมีส่วนเกี่ยวข้อง? | กระทรวง สปสช. ภาคประชาสังคม |

## 3. การวิเคราะห์แต่ละส่วน

### 3.1 Content (เนื้อหานโยบาย)

**คำถามสำคัญ:**
1. นโยบายว่าด้วยเรื่องใด?
2. เป้าหมายและวัตถุประสงค์คืออะไร?
3. กลยุทธ์และแนวทางการดำเนินงานคืออะไร?
4. ตัวชี้วัดความสำเร็จคืออะไร?

**ตัวอย่าง: หลักประกันสุขภาพถ้วนหน้า (UCS)**

| องค์ประกอบ | รายละเอียด |
|-----------|-----------|
| เรื่อง | สิทธิการรักษาพยาบาลสำหรับทุกคน |
| เป้าหมาย | ครอบคลุมประชากร 100% |
| กลยุทธ์ | สปสช. จ่ายตามผลงาน (Capitation) |
| ตัวชี้วัด | Coverage, Catastrophic expenditure |

### 3.2 Context (บริบท)

**บริบท 4 ประการ:**

```
┌─────────────────────────────────────────────────────────────┐
│                      CONTEXT                                │
├─────────────┬─────────────┬─────────────┬─────────────────┤
│   Political │  Economic   │   Social    │   Technological │
│  (การเมือง) │  (เศรษฐกิจ) │   (สังคม)   │   (เทคโนโลยี)   │
├─────────────┼─────────────┼─────────────┼─────────────────┤
│ • รัฐบาล    │ • GDP       │ • วัฒนธรรม  │ • นวัตกรรม      │
│ • พรรคการ   │ • งบประมาณ  │ • ค่านิยม   │ • ระบบสารสนเทศ  │
│   เมือง     │ • หนี้สาธารณะ│ • ความเชื่อ │ • เทคโนโลยี     │
│ • การเลือก  │ • การลงทุน  │ • ความรู้   │   การแพทย์      │
│   ตั้ง      │             │             │                 │
└─────────────┴─────────────┴─────────────┴─────────────────┘
```

**ตัวอย่าง: UCS**

| บริบท | ปัจจัย | ผลต่อนโยบาย |
|------|--------|-----------|
| Political | รัฐบาลไทยรักไทย (2544) | ผลักดันเป็นวาระหลัก |
| Economic | วิกฤตเศรษฐกิ 2540 | ต้องประหยัดงบประมาณ |
| Social | ความเหลื่อมล้ำด้านสุขภาพ | เรียกร้องความเป็นธรรม |
| Technological | ระบบ IT ที่ดีขึ้น | จัดการข้อมูลได้ดี |

### 3.3 Process (กระบวนการ)

**ขั้นตอนใน Process:**

```
1. Agenda Setting (การตั้งวาระ)
          ↓
2. Policy Formulation (การร่างนโยบาย)
          ↓
3. Decision Making (การตัดสินใจ)
          ↓
4. Implementation (การนำไปปฏิบัติ)
          ↓
5. Evaluation (การประเมินผล)
```

**ตัวอย่าง: UCS**

| ขั้นตอน | รายละเอียด | Actors ที่เกี่ยวข้อง |
|---------|----------|-------------------|
| Agenda Setting | ปัญหาค่าใช้จ่ายสุขภาพสูง | ภาคประชาชน, นักวิชาการ |
| Formulation | ออกแบบระบบ 30 บาทรักษาทุกโรค | สำนักงานปลัด สธ. |
| Decision | คณะรัฐมนตรีอนุมัติ | รัฐบาล |
| Implementation | สปสช. ดำเนินการ | สปสช., หน่วยบริการ |
| Evaluation | ประเมินผลทุกปี | IHPP, สวรส. |

### 3.4 Actors (ผู้มีส่วนได้ส่วนเสีย)

**ประเภทของ Actors:**

| ประเภท | ตัวอย่าง | บทบาท |
|--------|---------|-------|
| **State Actors** | กระทรวงสาธารณสุข, สปสช. | กำหนดนโยบาย จัดสรรทรัพยากร |
| **Non-State Actors** | สมาคมแพทย์, มูลนิธิ | ให้คำแนะนำ ติดตามตรวจสอบ |
| **International** | WHO, World Bank | ให้ทุน ให้คำปรึกษา |
| **Private Sector** | รพ.เอกชน, บริษัทประกัน | ให้บริการ มีผลประโยชน์ |
| **Civil Society** | กลุ่มผู้ป่วย, ชุมชน | เรียกร้อง ร่วมตัดสินใจ |

**ตัวอย่าง: UCS**

```
                    รัฐบาล
                      |
          +-----------+-----------+
          |                       |
    กระทรวงสธ.                สปสช.
          |                       |
    +-----+-----+           +-----+-----+
    |           |           |           |
รพ.รัฐ    รพ.สต.      รพ.เอกชน   ผู้ป่วย
```

## 4. การใช้ Policy Triangle วิเคราะห์นโยบาย

### 4.1 ขั้นตอนการวิเคราะห์

```
ขั้นที่ 1: ระบุนโยบายที่จะวิเคราะห์
              ↓
ขั้นที่ 2: เก็บข้อมูล Content
              ↓
ขั้นที่ 3: วิเคราะห์ Context
              ↓
ขั้นที่ 4: ตรวจสอบ Process
              ↓
ขั้นที่ 5: ระบุ Actors
              ↓
ขั้นที่ 6: วิเคราะห์ความสัมพันธ์ระหว่างส่วนต่างๆ
              ↓
ขั้นที่ 7: สรุปผลและข้อเสนอแนะ
```

### 4.2 ตัวอย่างการวิเคราะห์: นโยบายควบคุมยาสูบ

| ส่วน | รายละเอียด |
|-----|-----------|
| **Content** | พ.ร.บ.ควบคุมผลิตภัณฑ์ยาสูบ พ.ศ. 2560 |
| | • ห้ามโฆษณา |
| | • เพิ่มภาษี |
| | • คำเตือนบนซอง |
| **Context** | |
| - Political | รัฐบาลให้ความสำคัญ (ลงนาม FCTC) |
| - Economic | รายได้จากภาษาสูง แต่ต้นทุนโรคสูงกว่า |
| - Social | คนไทยสูบบุหรี่ 10 ล้านคน |
| - Technological | บุหรี่ไฟฟ้าใหม่ |
| **Process** | ผลักดานาน 10 ปี ต่อสู้กับอุตสาหกรรม |
| **Actors** | |
| - สนับสนุน | สธ., สสส., ภาคประชาสังคม |
| - คัดค้าน | อุตสาหกรรมยาสูบ |
| - เป็นกลาง | กระทรวงการคลัง (เก็บภาษี) |

### 4.3 ตัวอย่างการวิเคราะห์: นโยบายยกระดับบริการปฐมภูมิ

| ส่วน | รายละเอียด |
|-----|-----------|
| **Content** | โครงการก้าวทันความเจ็บป่วย |
| | • ลดความแออัด รพ. |
| | • เพิ่มบทบาท รพ.สต. |
| | • Capitation 30% ให้ รพ.สต. |
| **Context** | |
| - Political | รัฐบาลเน้น Universal Health Coverage |
| - Economic | ต้นทุนผู้ป่วยนอกสูง |
| - Social | คนไป รพ.ใหญ่ไม่จำเป็น |
| - Technological | Telemedicine |
| **Process** | เปลี่ยนระบบการเงิน ต้องมีการเจรจา |
| **Actors** | |
| - สนับสนุน | สธ., สปสช., ผู้ป่วย |
| - คัดค้าน | รพ.ใหญ่ (กลัวรายได้ลด) |
| - เป็นกลาง | แพทย์สมาคม |

## 5. การวิเคราะห์ความสัมพันธ์ระหว่างส่วนต่างๆ

### 5.1 ความสัมพันธ์สำคัญ

```
Content ↔ Context: เนื้อหาสอดคล้องกับบริบทหรือไม่?
Content ↔ Process: กระบวนการเหมาะสมกับเนื้อหาหรือไม่?
Content ↔ Actors: Actors เห็นด้วยกับเนื้อหาหรือไม่?
Context ↔ Process: บริบทเอื้อต่อกระบวนการหรือไม่?
Context ↔ Actors: Actors มีอำนาจในบริบทนี้หรือไม่?
Process ↔ Actors: Actors เข้าร่วมกระบวนการหรือไม่?
```

### 5.2 ตัวอย่างความสัมพันธ์

**กรณี: UCS สำเร็จ**
- Content ตอบโจทย์ Context (ความเหลื่อมล้ำ)
- Process มี Actors สำคัญร่วม (สธ., สปสช.)
- Context Political เอื้ออำนวย (รัฐบาลสนับสนุน)

**กรณี: นโยบายล้มเหลว**
- Content ไม่สอดคล้อง Context (เทคโนโลยีไม่พร้อม)
- Process ไม่มี Actors สำคัญร่วม (ไม่ consult ผู้ป่วย)
- Context Political ไม่เอื้อ (เปลี่ยนรัฐบาลบ่อย)
