# Workshop 4.1: Problem Statement Writing
## แบบฝึกหัด: การเขียน Problem Statement อย่างมีประสิทธิภาพ

---

## คำสั่ง
เลือก **หนึ่งปัญหา** จากรายการด้านล่าง แล้วเขียน Problem Statement ให้สมบูรณ์ตามโครงสร้าง

### ปัญหาที่เลือก (เลือกเพียงหนึ่งข้อ):
- [ ] อัตราตายมารดาสูงในพื้นที่ห่างไกล
- [ ] โรคเบาหวานไม่ควบคุมในกลุ่มผู้สูงอายุ
- [ ] การขาดแคลนแพทย์ในภาคตะวันออกเฉียงเหนือ
- [ ] อัตราการสูบบุหรี่ในวัยรุ่นเพิ่มขึ้น
- [ ] การเข้าถึงบริการสุขภาพของผู้พิการ
