# Module 4: การสร้างข้อเสนอแนะเชิงหลักฐาน

## หัวข้อการสอน

### 4.1 โครงสร้างข้อเสนอแนะนโยบาย (3 ชั่วโมง)
- ส่วนประกอบของข้อเสนอแนะที่ดี (7 ส่วน)
- การเขียน Problem Statement
- Theory of Change และ Logic Model

### 4.2 การเขียนอย่างมีประสิทธิภาพ (3 ชั่วโมง)
- หลักการเขียนเชิงนโยบาย
- การใช้ข้อมูลสนับสนุน
- SMART Recommendations
- การจัดการข้อจำกัด

### 4.3 การนำเสนอและสื่อสารนโยบาย (2 ชั่วโมง)
- การนำเสนอตามกลุ่มเป้าหมาย
- เทคนิคการสื่อสาร (Storytelling, Visualization)
- การสร้างพันธมิตรเพื่อผลักดันนโยบาย

## โครงสร้างข้อเสนอแนะที่ดี

```
1. Executive Summary (บทสรุปผู้บริหาร)
   - ปัญหา สาเหตุหลัก ข้อเสนอแนะ ประโยชน์

2. Problem Analysis
   - Problem Statement
   - Root Causes
   - หลักฐานข้อมูล

3. Stakeholder Analysis
   - ผู้มีส่วนได้ส่วนเสีย
   - กลยุทธ์การมีส่วนร่วม

4. Policy Options Analysis
   - ทางเลือกที่พิจารณา
   - การเปรียบเทียบ

5. Recommended Policy
   - ทางเลือกที่แนะนำ
   - Theory of Change

6. Implementation Plan
   - Timeline
   - งบประมาณ
   - Risk Assessment

7. Monitoring & Evaluation
   - KPIs
   - LogFrame
```

## SMART Recommendations

| ตัวอักษร | ความหมาย | ตัวอย่าง |
|---------|---------|---------|
| S | Specific | เฉพาะเจาะจง |
| M | Measurable | วัดได้ |
| A | Achievable | เป็นไปได้ |
| R | Relevant | สัมพันธ์กับปัญหา |
| T | Time-bound | มีกรอบเวลา |
