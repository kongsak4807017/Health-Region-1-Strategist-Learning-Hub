**เรียนรู้การใช้ข้อมูลสุขภาพไทยอย่างมีประสิทธิภาพ**

|------|-----------|

| M1-Workshop-Data-Verification.md | Workshop ตรวจสอบคุณภาพข้อมูล |
| M1-Worksheet-Data-Gap-Analysis.md | แบบฝึกวิเคราะห์ช่องโหว่ข้อมูล |
| M1-Sample-HDC-Data.csv | ตัวอย่างข้อมูล HDC |
| M1-Sample-NHSO-Data.csv | ตัวอย่างข้อมูล NHSO |
| M1-Sample-SES-Data.csv | ตัวอย่างข้อมูล SES |

| M1-Template-Data-Gap-Analysis.xlsx | Template วิเคราะห์ช่องโหว่ |
| M1-Checklist-TRUST.md | Checklist ประเมินคุณภาพข้อมูล |

**Learning Outcomes:**
- รู้จักแหล่งข้อมูลสุขภาพไทยหลัก 18+ แหล่ง

### Module 5: Capstone Project (8 ชั่วโมง)

## วิธีใช้เอกสาร

### สำหรับผู้สอน
1. อ่าน README ของแต่ละโมดูลก่อนเริ่ม
2. พิมพ์ Handouts สำหรับผู้เรียน
3. เตรียม Worksheets สำหรับกิจกรรม
4. ใช้ Case Studies เป็นการบ้านหรือตัวอย่าง

### สำหรับผู้เรียน
1. อ่าน Handout ก่อนเข้าชั้นเรียน
2. ทำ Worksheet ระหว่างเรียน
3. ศึกษา Case Study เพิ่มเติม
4. เก็บเอกสารเป็น Portfolio
