# Module 5: Capstone Project
## โครงการบูรณาการ: การวิเคราะห์และเสนอนโยบายสุขภาพเต็มรูปแบบ

---

## ภาพรวม

Module 5 เป็นโครงการบูรณาการสุดท้ายของหลักสูตร Thai Health Strategist ผู้เรียนจะนำความรู้และทักษะจาก Module 1-4 มาพัฒนาโครงการวิเคราะห์และเสนอนโยบายสุขภาพแบบเต็มรูปแบบ ตั้งแต่การรวบรวมข้อมูล วิเคราะห์สาเหตุรากเหง้า วิเคราะห์นโยบาย ไปจนถึงการสร้างข้อเสนอแนะและสื่อสารนโยบาย

**ระยะเวลา:** 8 ชั่วโมง (1 วัน)  
**รูปแบบ:** Workshop + Presentation  
**ผลลัพธ์:** Policy Analysis Report + Policy Brief + การนำเสนอ

## โจทย์โครงการ

### ตัวเลือกปัญหา (เลือก 1 จาก 5)

#### Option A: อัตราตายมารดาสูงในพื้นที่ห่างไกล
**บริบท:** อำเภอแม่สะเรียง จังหวัดแม่ฮ่องสอน  
**ข้อมูลที่ให้:** สถิติอัตราตายมารดา 5 ปี แผนที่บริการ ข้อมูลบุคลากร

#### Option B: โรคเบาหวานไม่ควบคุมในเขตเมือง
**บริบท:** กรุงเทพมหานคร  
**ข้อมูลที่ให้:** ข้อมูลผู้ป่วย DM จาก NHSO ข้อมูลพฤติกรรมจากสำรวจ

#### Option C: การขาดแคลนแพทย์ในภาคตะวันออกเฉียงเหนือ
**บริบท:** จังหวัดอุบลราชธานีและศรีสะเกษ  
**ข้อมูลที่ให้:** ข้อมูลบุคลากร การกระจาย อัตราการลาออก

#### Option D: อุบัติเหตุทางถนนในช่วงเทศกาล
**บริบท:** จังหวัดภาคเหนือตอนบน  
**ข้อมูลที่ให้:** ข้อมูลอุบัติเหตุ 7 วันอันตราย 3 ปี

#### Option E: สุขภาพจิตวัยรุ่น
**บริบท:** โรงเรียนมัธยมศึกษาในจังหวัดเชียงใหม่  
**ข้อมูลที่ให้:** ผลสำรวจสุขภาพจิตนักเรียน ข้อมูลบริการ

## ผลงานที่ต้องส่ง

### 1. Policy Analysis Report (70%)
รายงานวิเคราะห์นโยบายฉบับสมบูรณ์ ประกอบด้วย:

```
สารบัญ
บทที่ 1: บทนำ (5%)
   - ที่มาและความสำคัญ
   - วัตถุประสงค์
   - ขอบเขตการศึกษา

บทที่ 2: การวิเคราะห์สถานการณ์ (20%)
   - Problem Statement
   - ข้อมูลสถิติ
   - การวิเคราะห์ช่องโหว่ข้อมูล

บทที่ 3: การวิเคราะห์สาเหตุ (20%)
   - RCA (5 Whys, Fishbone, Causal Loop)
   - สาเหตุระดับต่างๆ
   - Priority Problem

บทที่ 4: การวิเคราะห์นโยบาย (15%)
   - Policy Triangle
   - Stakeholder Analysis
   - Policy Options + MCDA

บทที่ 5: ข้อเสนอแนะ (25%)
   - Recommended Policy
   - Theory of Change
   - Implementation Plan
   - Monitoring & Evaluation

บทที่ 6: สรุปและข้อเสนอแนะ (5%)

ภาคผนวก
   - ข้อมูลแหล่งที่มา
   - รายละเอียดเพิ่มเติม
   - References
```

### 2. Policy Brief (20%)
- ความยาว: 2-4 หน้า A4
- ภาษา: ไทย
- เป้าหมาย: ผู้บริหารสาธารณสุขระดับสูง

### 3. การนำเสนอ (10%)
- ระยะเวลา: 10 นาที + 5 นาที Q&A
- สไลด์: ไม่เกิน 12 สไลด์
- สมาชิกนำเสนอ: ทุกคนในกลุ่มต้องนำเสนอ

## คำแนะนำสำหรับผู้เรียน

### ก่อนเริ่มโครงการ
1. ทบทวนเอกสาร Module 1-4
2. เลือกปัญหาที่สนใจและมีข้อมูลพอ
3. อ่านข้อมูล background ที่ให้มา

1. แบ่งงานให้ชัดเจน ทุกคนต้องมีส่วนร่วม
2. ใช้เวลาให้เหมาะสมกับแต่ละส่วน
3. ปรึกษาผู้เชี่ยวชาญเมื่อติดขัด

### ก่อนนำเสนอ
1. ซ้อมนำเสนออย่างน้อย 1 ครั้ง
2. เตรียมตอบคำถามที่เป็นไปได้
3. ตรวจสอบสไลด์ให้เรียบร้อย
