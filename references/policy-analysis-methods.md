# Health Policy Analysis Methods

## กรอบการวิเคราะห์นโยบาย (Policy Analysis Frameworks)

### 1. Policy Triangle Framework (Walt & Gilson)

```
        Content (เนื้อหา)
           /\
          /  \
         /    \
        /______\
Context (บริบท)  Process (กระบวนการ)
```

**Content (เนื้อหานโยบาย)**
- นโยบายว่าด้วยเรื่องใด
- เป้าหมายและวัตถุประสงค์
- กลยุทธ์และแนวทางการดำเนินงาน
- ตัวชี้วัดความสำเร็จ

**Context (บริบท)**
- บริบทการเมือง: รัฐบาล การเมือง อำนาจตัดสินใจ
- บริบทเศรษฐกิจ: งบประมาณ การลงทุน
- บริบทสังคม: วัฒนธรรม ค่านิยม
- บริบทเทคโนโลยี: ความก้าวหน้าทางวิทยาศาสตร์

**Process (กระบวนการ)**
- การร่างนโยบาย
- การตัดสินใจและผลักดัน
- การนำไปปฏิบัติ
- การติดตามและประเมินผล

**Actors (ผู้มีส่วนได้ส่วนเสีย)**
- ภาครัฐ: กระทรวงสาธารณสุข สปสช. หน่วยงานที่เกี่ยวข้อง
- ภาคเอกชน: โรงพยาบาลเอกชน บริษัทประกัน
- ภาคประชาสังคม: มูลนิธิ สมาคมผู้ป่วย
- ประชาชน: กลุ่มเป้าหมาย ผู้รับบริการ

### 2. Health in All Policies (HiAP) Framework

แนวคิดที่ให้ทุกภาคส่วนพิจารณาผลกระทบทางสุขภาพ

**หลักการ 5 ประการ:**
1. ส่งเสริมสุขภาพและความเท่าเทียมกัน
2. ความร่วมมือระหว่างภาคส่วน
3. ประโยชน์ร่วมกัน (Win-Win)
4. การมีส่วนร่วมของประชาชน
5. ความโปร่งใสและตรวจสอบได้

**การประเมินผลกระทบทางสุขภาพ (Health Impact Assessment)**
- ระบุนโยบายที่อาจมีผลกระทบต่อสุขภาพ
- วิเคราะห์ประชากรที่ได้รับผลกระทบ
- ประเมินผลกระทบเชิงบวกและลบ
- เสนอแนะแนวทางลดผลกระทบลบ

### 3. Policy Cycle Framework

```
1. Agenda Setting → 2. Policy Formulation → 3. Decision Making → 
4. Implementation → 5. Evaluation → (back to 1)
```

**Stage 1: Agenda Setting**
- ปัญหานี้สำคัญพอที่จะขึ้น agenda หรือไม่
- ใครเป็นผู้ผลักดันให้ปัญหาขึ้น agenda
- Windows of opportunity: โอกาสที่เหมาะสม

**Stage 2: Policy Formulation**
- ระดมทางเลือกนโยบาย
- วิเคราะห์ต้นทุน-ผลประโยชน์
- คาดการณ์ผลที่ตามมา

**Stage 3: Decision Making**
- กระบวนการตัดสินใจของรัฐ
- การเจรจาและประนีประนอม
- ปัจจัยที่มีอิทธิพลต่อการตัดสินใจ

**Stage 4: Implementation**
- หน่วยงานรับผิดชอบ
- ทรัพยากรที่ต้องใช้
- อุปสรรคที่อาจเกิดขึ้น

**Stage 5: Evaluation**
- ประเมินผลลัพธ์ (outcomes)
- ประเมินผลกระทบ (impacts)
- ประเมินประสิทธิภาพ (efficiency)

### 4. Stakeholder Analysis Matrix

| Stakeholder | Interest | Influence | Position | Strategy |
|-------------|----------|-----------|----------|----------|
| กระทรวงสาธารณสุข | สูง | สูง | สนับสนุน | ร่วมมือ |
| สปสช. | สูง | สูง | เป็นกลาง | โน้มน้าว |
| รพ.เอกชน | ปานกลาง | ปานกลาง | คัดค้าน | เจรจา |
| ผู้ป่วย | สูง | ต่ำ | สนับสนุน | ให้ข้อมูล |

**การวิเคราะห์ผู้มีส่วนได้ส่วนเสีย:**
1. ระบุผู้มีส่วนได้ส่วนเสียทั้งหมด
2. วิเคราะห์ผลประโยชน์ (Interest)
3. วิเคราะห์อำนาจต่อรอง (Influence)
4. กำหนดกลยุทธ์การมีส่วนร่วม

### 5. SWOT Analysis for Policy Options

| | Positive | Negative |
|---|----------|----------|
| Internal | **Strengths** (จุดแข็ง) | **Weaknesses** (จุดอ่อน) |
| External | **Opportunities** (โอกาส) | **Threats** (อุปสรรค) |

**ตัวอย่างการใช้ SWOT วิเคราะห์นโยบาย:**

นโยบาย: ยกระดับคุณภาพการบริการปฐมภูมิ

**Strengths:**
- มีโครงสร้าง รพ.สต. ครอบคลุมทั่วประเทศ
- มีบุคลากรด้าน PHC จำนวนมาก

**Weaknesses:**
- บุคลากรขาดแคลนในพื้นที่ห่างไกล
- อุปกรณ์และยาไม่เพียงพอ

**Opportunities:**
- นโยบายรัฐบาลเน้นการสาธารณสุข
- เทคโนโลยี Telemedicine

**Threats:**
- งบประมาณจำกัด
- การย้ายถิ่นของบุคลากรไป รพ.เอกชน

### 6. Cost-Effectiveness Analysis (CEA)

การวิเคราะห์ต้นทุนต่อประสิทธิผล

**ICER (Incremental Cost-Effectiveness Ratio):**
```
ICER = (Cost_new - Cost_old) / (Effect_new - Effect_old)
```

**ประเมินความคุ้มค่าในบริบทไทย:**
- เกณฑ์ WTP (Willingness to Pay): ~160,000 THB/QALY
- ถ้า ICER < WTP → นโยบายคุ้มค่า

### 7. Logic Model for Program Evaluation

```
Inputs → Activities → Outputs → Outcomes → Impacts
(ทรัพยากร) (กิจกรรม) (ผลผลิต) (ผลลัพธ์) (ผลกระทบ)
```

**ตัวอย่าง Logic Model โครงการลดการสูบบุหรี่:**

| Inputs | Activities | Outputs | Outcomes | Impacts |
|--------|------------|---------|----------|---------|
| งบประมาณ | รณรงค์ | จำนวนกิจกรรม | ความรู้เพิ่มขึ้น | ลดอัตราสูบบุหรี่ |
| บุคลากร | อบรม | จำนวนผู้เข้าร่วม | เปลี่ยนทัศนคติ | ลดโรคปอด |
| สื่อ | สร้างกฎหมาย | จำนวนพื้นที่ | เปลี่ยนพฤติกรรม | ลดค่าใช้จ่าย รพ. |

### 8. Policy Diffusion Framework

การเผยแพร่นโยบายระหว่างประเทศ/พื้นที่

**รูปแบบการเผยแพร่:**
- **Global diffusion**: นโยบายระดับโลก (SDGs, WHO guidelines)
- **Regional diffusion**: นโยบายระดับภูมิภาค (ASEAN)
- **Horizontal diffusion**: เรียนรู้ระหว่างประเทศ (Best practices)
- **Vertical diffusion**: จากกลางสู่ท้องถิ่น

**การนำนโยบายมาประยุกต์ใช้ในบริบทไทย:**
1. ศึกษาประสบการณ์ต่างประเทศ
2. วิเคราะห์บริบทความเหมาะสม
3. ปรับแต่งให้เหมาะสมกับบริบทไทย
4. ทดลองนำร่อง (Pilot)
5. ขยายผล

## เทคนิคการเปรียบเทียบทางเลือกนโยบาย

### Multi-Criteria Decision Analysis (MCDA)

**ตัวอย่างเกณฑ์ประเมิน:**
1. Effectiveness (ประสิทธิผล)
2. Cost (ต้นทุน)
3. Equity (ความเป็นธรรม)
4. Feasibility (ความเป็นไปได้)
5. Acceptability (ความยอมรับ)

**การให้คะแนน:**
- ให้น้ำหนักแต่ละเกณฑ์ (รวม = 100%)
- ให้คะแนนแต่ละทางเลือก (1-5)
- คำนวณคะแนนรวม

### Scenario Planning

การวางแผนรองรับหลายสถานการณ์

**สถานการณ์ที่พิจารณา:**
1. Best case: สถานการณ์ที่ดีที่สุด
2. Most likely: สถานการณ์ที่เป็นไปได้มากที่สุด
3. Worst case: สถานการณ์แย่ที่สุด

## การวิเคราะห์นโยบายในระบบสุขภาพไทย

### พิจารณาบริบทเฉพาะของไทย

**จุดแข็งของระบบสุขภาพไทย:**
- หลักประกันสุขภาพถ้วนหน้า (UCS)
- โครงสร้าง รพ.สต. ครอบคลุมทุกตำบล
- บุคลากรสาธารณสุขจำนวนมาก

**ความท้าทาย:**
- สังคมสูงอายุ
- ภาระโรค NCD เพิ่มขึ้น
- ความเหลื่อมล้ำด้านสุขภาพ
- การขาดแคลนบุคลากรในพื้นที่ห่างไกล

### การวิเคราะห์กฎหมายและระเบียบที่เกี่ยวข้อง

- พระราชบัญญัติหลักประกันสุขภาพแห่งชาติ
- พระราชบัญญัติสถานพยาบาล
- พระราชบัญญัติควบคุมโรคติดต่อ
- ระเบียบกระทรวงสาธารณสุขที่เกี่ยวข้อง
